Test ClientId Header Functionality

Method 1: Browser Developer Tools

Open your browser's developer tools and use the Network tab to add custom headers:

  1. Open Developer Tools (F12)
  2. Go to Network tab
  3. Right-click on any request and select "Copy as cURL"
  4. Add the header: -H "clientId: test"

Method 3: Using cURL

Test from command line:

# With clientId=test (should return dark theme)
curl -H "clientId: test" http://localhost:3001/api/check-client-id

# Without clientId (should return default theme)
curl http://localhost:3001/api/check-client-id

Method 4: Browser Extension

Use a browser extension like "ModHeader" to add custom headers to all requests:

  1. Install ModHeader extension
  2. Add header: clientId: test
  3. Refresh the page
